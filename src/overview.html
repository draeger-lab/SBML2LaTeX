SBML2LaTeX

SBML2LaTeX is a documentation tool for models of biological processes that are encoded in the Systems Biology Markup Language (SBML, see sbml.org and SBML2LaTeX for details).

This program can be used in multiple ways. The most common scenario would be to run it on a web server because for many users the installation of the LaTeX system can be tricky. If a LaTeX installation is available, a graphical user interface provides several options to customize the layout and information content of the model report to be generated by SBML2LaTeX. Another option would be to run SBML2LaTeX using its rich command-line interface or to embed it into third-party programs.

Since the application itself is very light-weight and simple, this API documentation is intended to help understanding how to use it and how to run SBML2LaTeX as a report-generator back-end in an end-user application.

Since version 1.0 SBML2LaTeX also supports the SBML Layout extension and can create drawings of biological networks. To this end, SBML2LaTeX interprets the layout description in the model so that the images look like Process Diagrams of the Systems Biology Graphical Notation (SBGN, see sbgn.org). The LaTeX package TikZ is used to draw the images inside of the LaTeX document, similar as this has been implemented in SBML2tikz.

For mor information about SBML2LaTeX, please see the project web site and please cite the publication about SBML2LaTeX.
